# Studious-bassoon
Características principales:  6 boxes con animación de bounce-in en cascada  Cubic-bezier personalizado para efecto de resorte  Delays escalonados (Os, 0.1s, 0.2s, etc.)  Colores degradados únicos para cada box  Botón para reiniciar la animación  Diseño responsive  Efectos hover interactivos
